# Anchor DAppNode Package

Anchor is an open source implementation of the Secret Shared Validator (SSV) protocol, written in Rust and maintained by [Sigma Prime](https://sigmaprime.io). This DAppNode package provides a user friendly UX to run Anchor as a distributed validator client on your DAppNode.

For more information about Anchor, refer to the official [documentation](https://anchor.sigmaprime.io/introduction).
